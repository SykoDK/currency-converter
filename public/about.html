About this project

This website was developed by Ian Grzebien

This is a simple currency tracker web app built with HTML, CSS, and JavaScript.

It uses the ExchangeRate-API to fetch live exchange rates and historical data.

It uses IPStack to determine user location.

It uses Supabase to store popular conversions

It uses Chart.js to display the historical data in a chart.

It uses Day.js to format the date and time.

It uses Fetch API to make requests to the APIs.

This web app shows live exchange rates and lets you convert currencies.
